# Admin Panel Guide

## Quick Start (For Basic Users)

### Just need to add a destination quickly?
1. Go to `Destinations`
2. Select `Quick add new destination` from the dropdown
3. Fill in:
   - **Name**: Destination name
   - **Region**: Select from list
   - **Short Description**: A brief intro
4. Click `Save`

## Destinations

### Adding a New Destination
1. Go to `Destinations` in the admin sidebar
2. Click `ADD DESTINATION +` button
3. Fill in the required fields:
   - **Name**: The full name of the destination
   - **Slug**: Auto-generated from the name (editable)
   - **Region**: Select from existing regions or add new
   - **Description**: Detailed information about the destination
   - **Short Description**: Brief summary (used in listings)
4. Add location details:
   - Address, City, Country, Postal Code
   - Latitude/Longitude (optional)
5. Set status:
   - ☑ Is Featured: Check to show on featured lists
   - ☑ Is Active: Check to make visible on site
6. Click `SAVE`

### Quick Add Destination
1. Go to `Destinations`
2. Select `Quick add new destination` from the action dropdown
3. Fill in the basic details
4. Click `Save and continue editing` to add more details

## Regions

### Adding a New Region
1. Go to `Regions` in the admin sidebar
2. Click `ADD REGION +`
3. Fill in:
   - **Name**: Region name (e.g., "North America")
   - **Slug**: Auto-generated from name
   - **Description**: Optional details about the region
4. Click `SAVE`

## Attractions

### Adding a New Attraction
1. Go to `Attractions` in the admin sidebar
2. Click `ADD ATTRACTION +`
3. Fill in:
   - **Name**: Attraction name
   - **Destination**: Select associated destination
   - **Category**: Choose from dropdown
   - **Description**: Detailed information
   - **Location**: Address, coordinates if available
   - **Contact**: Website, phone, email
4. Set status:
   - ☑ Is Featured: Check to highlight
5. Click `SAVE`

## Seasons

### Adding Season Information
1. Go to `Seasons` in the admin sidebar
2. Click `ADD SEASON +`
3. Select the destination
4. Fill in:
   - **Name**: Season name (e.g., "Summer", "Peak Season")
   - **Start Date** and **End Date**
   - ☑ Is Peak Season: Check if this is a high season
   - **Description**: Special notes or activities
5. Click `SAVE`

## Images

### Adding Destination Images
1. Go to `Destination images`
2. Click `ADD DESTINATION IMAGE +`
3. Select the destination
4. Upload an image file
5. Add a caption if needed
6. Set as featured image if desired
7. Click `SAVE`

### Adding Attraction Images
1. Go to `Attraction images`
2. Click `ADD ATTRACTION IMAGE +`
3. Select the attraction
4. Upload an image file
5. Add a caption if needed
6. Click `SAVE`

## Advanced Features

### Bulk Actions
1. Select multiple items using checkboxes
2. Choose an action from the dropdown
3. Click "Go" to apply to all selected items

### Export/Import Data
1. Go to the desired section (e.g., Destinations)
2. Click "Export" to download data as CSV/JSON
3. Use "Import" to upload data in bulk

### Custom Filters
1. Use the filter sidebar to create custom views
2. Save frequently used filters for quick access

## User Roles & Permissions

### Administrator
- Full access to all features
- Can manage users and permissions
- Access to system settings

### Editor
- Can create/edit all content
- Cannot modify user accounts
- Limited access to settings

### Author
- Can only edit their own content
- Cannot publish without approval
- Basic access only

## Common Issues

### Image Upload Problems
- Check file size (max 2MB recommended)
- Use common formats: JPG, PNG, WEBP
- Ensure proper file permissions on the server

### Form Validation Errors
- Red borders indicate required fields
- Hover over error icons for details
- Check that dates are in the correct format

### Saving Issues
- Check for required fields (marked with *)
- Ensure no special characters in slugs
- Verify internet connection if using external services
- Clear browser cache if forms don't load properly
